# Web vật tư CKQN - Supabase + Google Drive OAuth

- Supabase lưu dự án, vật tư, tình trạng và token OAuth.
- Google Drive lưu thư mục dự án và file Excel/PDF gốc.
- Google Drive kết nối bằng OAuth, không dùng Service Account.

Chạy file `SUPABASE_SCHEMA.sql` trong Supabase SQL Editor trước khi dùng.
