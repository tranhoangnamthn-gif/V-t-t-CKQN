Danh sách dự án

Tạo dự án trên web, hệ thống sẽ tự tạo thư mục tương ứng trên Google Drive. Sau đó upload Excel để import vật tư và lưu file gốc vào đúng thư mục dự án.

Tạo dự án mới

Quy trình sử dụng

  1. Tạo dự án, Google Drive sẽ tự tạo thư mục dự án.
  2. Upload file Excel trong trang dự án, file gốc tự lưu vào Google Drive.
  3. Hệ thống tự import các dòng vật tư.
  4. Mọi người tích trạng thái: Chưa đặt hàng / Đã đặt hàng.
  5. Xuất CSV để tổng hợp báo cáo khi cần.
